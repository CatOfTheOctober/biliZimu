# B站视频字幕提取工具 - 使用指南

## 🚀 快速开始（推荐方式）

### 使用 `下载字幕.py` 脚本

这是最简单、最稳定的使用方式，适合下载单个或多个视频的字幕。

#### 1. 准备环境
```bash
# 确保已安装 Python 3.8+
python --version

# 安装依赖
pip install -e .
```

#### 2. 准备 Cookie（重要！）
为了获取 AI 字幕，需要 B 站登录 Cookie：

```bash
# 进入 tools/BBDown 目录
cd tools/BBDown

# 运行 BBDown 登录（会自动保存 Cookie）
BBDown.exe --login

# Cookie 会自动保存到 BBDown.data 文件
```

#### 3. 下载字幕
```bash
# 直接运行下载脚本（交互式）
python 下载字幕.py
```

然后按提示输入：
- 完整 URL: `https://www.bilibili.com/video/BV1M8c7zSEBQ/`
- 短链接: `https://b23.tv/xxxxx`
- 直接 BVID: `BV1M8c7zSEBQ`

脚本会自动下载字幕并保存为：
- `output/视频标题_BVID.txt` （纯文本格式）
- `output/视频标题_BVID.srt` （SRT 格式，带时间戳）

## 🔧 备用方式

### 方式一：使用命令行工具

> ⚠️ 注意：命令行工具目前可能遇到 WBI 签名 412 错误，推荐使用上面的 `下载字幕.py` 脚本。

```bash
# 检查 Cookie 状态
bilibili-extractor --check-cookie

# 使用 BBDown 登录
bilibili-extractor --login

# 提取字幕
bilibili-extractor "https://www.bilibili.com/video/BV1M8c7zSEBQ/"

# 使用配置文件
bilibili-extractor "VIDEO_URL" --config config/config.yaml
```

### 方式二：Python 代码使用
```python
from bilibili_extractor.modules.bilibili_api import BilibiliAPI

# 加载 Cookie
with open('tools/BBDown/BBDown.data', 'r', encoding='utf-8') as f:
    cookie = f.read().strip()

# 创建 API 实例
api = BilibiliAPI(cookie=cookie)

# 获取字幕
result = api.get_subtitle_with_ai_fallback("BV1M8c7zSEBQ", cid)

if result['success']:
    # TXT 格式字幕（默认）
    txt_content = result['subtitle_text']
    
    # SRT 格式字幕
    srt_content = result['subtitle_text_srt']
    
    # 字幕列表
    subtitles = result['subtitles']
    
    print(f"获取到 {len(subtitles)} 条字幕")
```

## 📋 支持的功能

### ✅ 字幕类型
- **AI 字幕**：B 站自动生成的字幕（需要 Cookie）
- **官方字幕**：UP 主上传的字幕
- **多语言支持**：中文、英文、日语、西班牙语、阿拉伯语、葡萄牙语

### ✅ 输出格式
- **TXT 格式**：纯文本，每行一句话（默认）
- **SRT 格式**：标准字幕格式，带时间戳
- **JSON 格式**：原始数据格式

### ✅ 高级功能
- **智能降级**：优先获取 AI 字幕，失败时自动降级
- **缓存机制**：避免重复请求，提高效率
- **速率限制**：防止触发 B 站风控
- **详细日志**：完整的调试信息

## 🛠️ 自定义使用

### 修改目标视频
编辑 `download_subtitle_final.py` 文件：
```python
def main():
    # 修改这里的 URL
    video_url = "你的视频URL"
    
    filename = download_subtitle(video_url)
```

### 批量下载
创建批量下载脚本：
```python
video_urls = [
    "https://www.bilibili.com/video/BV1M8c7zSEBQ/",
    "https://www.bilibili.com/video/BV1234567890/",
    # 添加更多视频 URL
]

for url in video_urls:
    try:
        filename = download_subtitle(url)
        print(f"✅ 成功: {filename}")
    except Exception as e:
        print(f"❌ 失败: {url} - {e}")
```

### 只保存特定格式
如果只想要 TXT 格式，修改 `download_subtitle_final.py`：

## 🔧 故障排除

### 问题 1：没有找到字幕
**原因**：视频可能没有字幕，或需要 Cookie
**解决**：
1. 确认视频确实有字幕
2. 检查 Cookie 是否有效
3. 尝试重新登录获取 Cookie

### 问题 2：Cookie 加载失败
**原因**：Cookie 文件不存在或格式错误
**解决**：
1. 检查 `tools/BBDown/BBDown.data` 文件是否存在
2. 重新运行 BBDown 登录
3. 手动获取并保存 Cookie

### 问题 3：API 请求失败
**原因**：网络问题或被风控
**解决**：
1. 检查网络连接
2. 等待一段时间后重试
3. 检查 Cookie 是否过期

### 问题 4：字幕内容为空
**原因**：字幕 URL 无效或下载失败
**解决**：
1. 检查视频是否为付费内容
2. 尝试不同的视频
3. 查看详细日志排查问题
